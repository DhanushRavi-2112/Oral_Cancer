VGG16 Architecture

VGG16 is a proven convolutional neural network architecture known for its simplicity and effectiveness. With 16 layers and 14.8M parameters, it provides reliable performance for medical image classification tasks.

  • Strengths: Fast inference, stable training, proven results
  • Best for: Quick analysis, high-volume processing
RegNetY-320 Architecture

RegNetY-320 represents modern efficient network design with only 3.2M parameters. It uses optimized building blocks and scaling strategies for excellent performance-to-size ratio.

  • Strengths: Efficient, modern design, low memory usage
  • Best for: Resource-constrained environments, mobile deployment
Our Recommendation

For the highest accuracy in oral cancer detection, we recommend using the Ensemble model which combines the strengths of both VGG16 and RegNetY-320, achieving 94.2% accuracy with uncertainty estimation.
